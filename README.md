# Color Picker

Interactive color generator based on RGB sliders, allowing users to visually create colors in real time.

## Features
- Three sliders representing **Red**, **Green**, and **Blue** values.
- Real-time preview box that updates as the user drags the sliders.
- Displays the current RGB value as text.

## Technologies
- HTML
- CSS
- JavaScript (DOM events)
